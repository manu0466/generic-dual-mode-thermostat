# Generic Dual Mode Thermostat

This is a modification of the [Generic Thermostat Helper](https://www.home-assistant.io/integrations/generic_thermostat/) that can be set to operate in heating or cooling mode at the same time.

## Installation

1. Add <https://github.com/manu0466/generic-dual-mode-thermostat> to your [HACS](https://hacs.xyz/) custom repositories.
2. Choose `Integration` from the category selection.
3. Click install.
4. Return to the Integrations page within HACS then click the `+ Explore & download repositories` button.
5. Search for `Generic Dual Mode Thermostat`, select it, then click `Download this repository with HACS`.
6. Restart Home Assistant to load the integration.
